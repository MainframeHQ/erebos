# @erebos/api-bzz-base

Base class for Swarm (BZZ) APIs interactions. Used by [`@erebos/api-bzz-browser`](../api-bzz-browser) and [`@erebos/api-bzz-node`](../api-bzz-node).

## Installation

```sh
yarn add @erebos/api-bzz-base
```
